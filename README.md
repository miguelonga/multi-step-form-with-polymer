# Multi step form with polymer

Tested with capybara

Use 2.4.0 ruby version

## RUN APP

Download git:

~~~
git clone https://github.com/miguelonga/multi-step-form-with-polymer.git
~~~

Install gems:

~~~
bundle install
~~~

Launch server:

~~~
ruby app.rb
~~~

Test:

~~~
rspec
~~~

View: localhost:4567
